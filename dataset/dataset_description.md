# Dataset Description

## Dataset Information

**Dataset Name:** Global Superstore Dataset

**Source:** Kaggle – Superstore Sales Analytics Dataset 2026
https://www.kaggle.com/datasets/nafeesalmahadi/superstore-sales-analytics-dataset-2026

The dataset contains approximately 51,290 transactions from a global superstore business.
Time Period: 2011-01-01 to 2014-12-31 (4 years)
Data Type: Structured tabular data

Key fields:

* category - Product category (Furniture/Office Supplies/Technology)
* city - Customer city
* country	- Customer country
* customer_id - Unique customer identifier
* customer_name - Customer full name
* discount - Discount rate applied (0-0.85 range)
* market - Market region (14 unique markets)
* order_date - Date when order was placed
* order_id - Unique order identifier
* order_priority - Order priority (Critical/High/Medium/Low)
* product_id - Unique product identifier
* product_name - Product description
* profit - Total profit amount in USD (Target)
* quantity - Number of items ordered (1-14 range)
* region - Geographic region (14 regions)
* sales - Total sales amount in USD
* segment - Customer segment (Consumer/Corporate/Home Office)
* ship_date - Date when order was shipped
* ship_mode - Shipping method (First Class/Second Class/Standard Class/Same Day)
* shipping_cost - Cost of shipping in USD
* state - Customer state
* sub_category - Product sub-category (17 unique)

## Project Usage

This dataset was used to perform:

* Sales Performance Analysis
* Profitability Analysis
* Discount Impact Analysis
* ABC Analysis
* BCG Analysis
* RFM Customer Segmentation
* Cohort Retention Analysis
* Logistics Performance Analysis
* Geographic Profitability Analysis
