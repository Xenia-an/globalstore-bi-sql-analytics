# Superstore Sales Analysis

## Project Overview

Этот проект посвящен анализу продаж международной розничной компании на основе данных о 40 000 заказов.

Цель проекта — выявить ключевые факторы роста выручки и прибыли, оценить эффективность скидок, проанализировать клиентскую базу и определить наиболее перспективные направления для развития бизнеса.

## Tools Used

* PostgreSQL
* SQL
* DBeaver
* Power BI

## Dataset

Global Superstore Dataset

Количество записей: 40 000 заказов

## Business Questions

1. Рассчитать ежемесячные темпы роста продаж (MoM Growth) и определить сезонные колебания.
2. Провести аудит ценообразования и оценить финансовую эффективность системы скидок.
3. Провести ABC-анализ товарной матрицы по вкладу подкатегорий в общую выручку.
4. Классифицировать продуктовый портфель по матрице BCG (сопоставление оборота и чистой маржи).
5. Провести RFM-сегментацию клиентской базы для приоритизации маркетинговых активностей.
6. Рассчитать годовой коэффициент удержания клиентов (Retention Rate) по историческим когортам.
7. Оценить операционную дисциплину логистических каналов и структуру тарифов доставки.
8. Провести гео-анализ эффективности международной экспансии в разрезе стран и рынков.

## Key Insights

* Продажи демонстрируют устойчивый рост и выраженную сезонность.
* Technology является лидером по выручке среди основных категорий.
* Высокие скидки напрямую связаны со снижением прибыльности.
* Southeast Asia показывает признаки маржинального давления из-за агрессивной скидочной политики.
* Небольшая группа клиентов формирует значительную часть выручки компании.
* Уровень удержания клиентов остается высоким, однако привлечение новых клиентов снижается.
* США, Китай и Индия являются наиболее прибыльными рынками.
* Турция и Нигерия демонстрируют отрицательную прибыльность.
