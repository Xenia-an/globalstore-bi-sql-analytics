# Superstore Sales Analysis

## Project Overview

В рамках этого проекта я провела анализ данных международного интернет-магазина с использованием PostgreSQL и Power BI.

Цель проекта — изучить продажи, прибыльность, клиентскую базу и эффективность различных направлений бизнеса, а также выявить возможности для повышения прибыли компании.

Проект выполнен как часть обучения Data Analytics и включает полный цикл аналитической работы: от проверки качества данных и SQL-анализа до визуализации результатов в Power BI и подготовки бизнес-рекомендаций.

## Dataset

Для анализа использовался датасет **Global Superstore Dataset**.

Подробное описание данных находится в файле:

`dataset/dataset_description.md`

Основные характеристики датасета:

- Количество записей: 51 290 заказов
- Период данных: 2011–2014 гг.
- География: международные продажи
- Основные показатели: продажи, прибыль, скидки, клиенты, доставка

## Business Questions

В рамках проекта были исследованы следующие вопросы:

1. Как менялись продажи и прибыль компании по месяцам?
2. Как скидки влияют на прибыльность бизнеса?
3. Какие категории и подкатегории товаров приносят основную выручку?
4. Какие товары наиболее эффективны с точки зрения продаж и прибыли?
5. Кто является наиболее ценными клиентами компании?
6. Насколько хорошо компания удерживает клиентов?
7. Насколько эффективно работает доставка?
8. Какие страны приносят наибольшую прибыль, а какие работают неэффективно?

## Key Insights

- За период 2011–2014 гг. продажи компании демонстрировали устойчивый рост.
- Категория Technology принесла более 3.7 млн$ выручки и стала лидером среди всех категорий товаров.
- Наиболее высокие продажи наблюдались в августе–сентябре и ноябре–декабре, что указывает на выраженную сезонность бизнеса.
- Средняя скидка для прибыльных заказов составила 1.8%, тогда как для убыточных — 46.5%, что подтверждает сильное влияние скидочной политики на финансовый результат.
- Подкатегория **Tables** показала отрицательную прибыль(-10932.11$) несмотря на высокий объем продаж.
- По результатам RFM-анализа около 16.6% клиентов вошли в сегмент наиболее ценных покупателей.
- США стали самым прибыльным рынком компании с прибылью более 320 тыс$.
- Турция и Нигерия продемонстрировали отрицательную прибыльность и требуют пересмотра коммерческой стратегии.
